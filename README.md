# FreeWayGame


*🎮 Compartilhando um pequeno projeto que desenvolvi durante um curso de **JavaScript da Alura**.*


O projeto é baseado no clássico **Freeway Game**, um jogo simples onde o objetivo é atravessar a rua desviando dos carros. Durante o desenvolvimento, personalizei alguns elementos visuais e ajustes na lógica para dar meu próprio toque ao jogo.


Foi um ótimo exercício para praticar **lógica de programação, manipulação com JavaScript e interatividade no navegador**.
